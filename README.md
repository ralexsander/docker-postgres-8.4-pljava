# PostgreSQL 8.4 with PL/Java

> This container is an installation of databse PostgreSQL 8.4 with PL/Java on Ubuntu 14.04.

## How to execute

```shell=
docker run ralexsander/postgres-8.4-pljava
```

This container exposes the port 5432.
